# 등차수열
  
초항이 a, 공차가 d인 수열 $a_n = d + a_{n-1} = a + d(n-1)$  
여기서 $a_n$과 $a_{n-1}$에서 본 문제와 작은 문제 관계를 볼 수 있다.

## 등차수열의 재귀 알고리즘
```
def seqRec(n):
  if(n == 1):
    return a
  else:
    return seqRec(n-1) + d
```

## 등차수열의 비재귀적 알고리즘
```
def seq(n):
  return a + d*(n-1)
```
